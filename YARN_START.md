# Starting Wahizza App with Yarn

## Quick Start Commands

### First Time Setup
```bash
cd /Users/joeltavarez/Desktop/WahizzaApp
yarn install
```

### Start Development Server
```bash
yarn start
```

# Or use the npm script directly:
yarn expo start
```

## Available Yarn Commands

- `yarn start` - Start Expo development server
- `yarn ios` - Start and open iOS simulator
- `yarn android` - Start and open Android emulator
- `yarn web` - Start and open in web browser

## After Starting

Once `yarn start` is running, you'll see:
- A QR code in the terminal
- Options to press keys:
  - Press `i` for iOS simulator
  - Press `a` for Android emulator
  - Press `w` for web browser

Or scan the QR code with the Expo Go app on your phone!
